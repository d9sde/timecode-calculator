TimeCode Calculator

version 1.2
Keys:
  • n – reset current field to zero
  • z – new line
  • f – change framerate
  • + / - – toggles the operator in the current line (alternatively, simply click on the operator)
  • Arrow keys – navigate between fields
Mouse:
  • Drag&Drop – click inside any result and drag it into any input field
  • Mouse wheel – increases/decreases the value in the current input field by 1 frame
Buttons:
  • + / - – toggles the operator
  • = – copies the adjacent result to the clipboard
  • xx FPS – switch between framerates
  • 24/99 h – toggles the 24-hour wrap-around on or off (for real-time codes)
Cursor:
  • If the cursor is positioned before a digit, typing will simply overwrite it.
  • If the cursor is at the very right of an input field, existing digits will shift to the left as you type (like on a calculator).
